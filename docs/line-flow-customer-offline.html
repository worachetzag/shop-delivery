โฟลว์การใช้งานผ่าน LINE

ไฟล์นี้เปิดได้จากคอมพิวเตอร์หรือมือถือโดยไม่ต้องมีอินเทอร์เน็ต — ส่งต่อเป็นไฟล์แนบหรือเก็บในเครื่องได้

1) เปิดจาก LINE

เริ่มลูกค้าแตะลิงก์ใน LINE (เมนู / ข้อความ / แจ้งเตือน)
↓
แบบ LIFF LINE พาเข้าหน้าร้านตามที่ตั้งในระบบ LINE
หรือ
แบบลิงก์เว็บ เปิดหน้าร้านในเบราว์เซอร์โดยตรง
↓
โหลดหน้าร้านออนไลน์
↓
ถ้ามีพารามิเตอร์จาก LINE (เช่น liff.state) ระบบจะพาไปหน้าที่ถูกต้องอัตโนมัติ
↓
จบเห็นหน้าสินค้า / ตะกร้า / รายการออเดอร์ / รายละเอียดออเดอร์ ตามที่เลือก

2) ที่อยู่มีคำว่า liff.state

LINE บางครั้งซ่อนชื่อหน้าในพารามิเตอร์นี้ ระบบจะถอดความหมายแล้วพาไป URL ชัด เช่น /customer/orders/7

ความหมายของหน้าอยู่ใน LINE
↓
ครั้งแรกในเบราว์เซอร์ที่อยู่อาจเป็น /customer/?liff.state=...
↓
หลังประมวลผลที่อยู่กลายเป็นรูปแบบปกติ เช่น /customer/orders/7

3) แจ้งเตือนสถานะคำสั่งซื้อ

ร้านอัปเดตสถานะออเดอร์
↓
ระบบส่งข้อความใน LINEถึงลูกค้า
↓
มีปุ่มหรือลิงก์ดูรายละเอียดออเดอร์
↓
ลูกค้าแตะเข้าหน้ารายละเอียดของออเดอร์ชุดนั้น

4) เข้าสู่ระบบด้วย LINE (ลำดับโดยย่อ)

  1. ลูกค้าเปิดหรือแตะ «เข้าสู่ระบบด้วย LINE»
  2. หน้าร้านติดต่อระบบร้านเพื่อเริ่มล็อกอิน
  3. LINE ให้ยืนยันตัวตน
  4. ระบบร้านยืนยันสำเร็จแล้วส่งกลับมาที่หน้าร้าน
  5. ลูกค้าใช้งานร้านในสถานะล็อกอินแล้ว
ซำสูงพาณิชย์ — เอกสารอธิบายเบื้องต้น สำหรับเปิดในเครื่องหรือแชร์เป็นไฟล์เท่านั้น
