คู่มือโฟลว์การใช้งานระบบ

สรุปภาพรวมและฟังก์ชันหลักของระบบร้านค้าออนไลน์และหลังบ้าน — เปิดได้จากไฟล์เดียวโดยไม่ต้องต่ออินเทอร์เน็ต
ซำสูงพาณิชย์ · Samsung Panich Delivery

สารบัญ

  • บทบาทผู้ใช้ในระบบ
  • โฟลว์หลักลูกค้า (สั่งซื้อสินค้า)
  • ฟังก์ชันแต่ละหน้า — ลูกค้า
  • การเชื่อมกับ LINE
  • คนขับจัดส่ง
  • แอดมินร้าน (หลังบ้าน)

บทบาทผู้ใช้ในระบบ

  • ลูกค้าสั่งซื้อผ่านหน้าเว็บหรือ LINE LIFF — ต้องเข้าสู่ระบบสำหรับตะกร้า เช็คเอาท์ ออเดอร์ โปรไฟล์
  • คนขับรับงานและอัปเดตสถานะการจัดส่งผ่านหน้าคนขับ
  • แอดมินร้านจัดการสินค้า ออเดอร์ คลัง บุคลากร และตั้งค่าร้านผ่านแดชบอร์ดบนเว็บ
  • ผู้ดูแลระบบฟีเจอร์ขั้นสูง เช่น audit log และการจัดการที่ขึ้นกับสิทธิ์ที่เปิดในระบบ

โฟลว์หลักลูกค้า (สั่งซื้อสินค้า)

ลำดับทั่วไปตั้งแต่เข้าร้านจนถึงได้รับสินค้า

1) เข้าร้าน → หน้าแรกแสดงโปรโมชันและสินค้าแนะนำ
↓
2) เลือกหมวด / ค้นหา → หน้ารายการสินค้า → เปิดรายละเอียดสินค้า → เพิ่มในตะกร้า
↓
3) เข้าสู่ระบบ (ถ้ายังไม่ล็อกอิน) → เปิดตะกร้า → ปรับจำนวน / ลบรายการ
↓
4) เช็คเอาท์ → เลือกที่อยู่จัดส่งหรือรับที่ร้าน → เลือกวิธีชำระเงินตามที่ร้านเปิดใช้ → ยืนยันคำสั่งซื้อ
↓
5) ติดตามในหน้า «ออเดอร์» → ดูรายละเอียด / แนบหลักฐานการโอน (ถ้ามี) → เปิดติดตามเมื่ออยู่ระหว่างจัดส่ง
↓
6) ได้รับสินค้า → สถานะเป็น «จัดส่งสำเร็จ» (หรือสถานะที่ร้านกำหนด)

ฟังก์ชันแต่ละหน้า — ลูกค้า

หน้าแรก /customer

  • ดูแบนเนอร์โปรโมชันและสินค้าตามหมวดที่แอดมินตั้งบนหน้าแรก
  • เลื่อนดูและกดเข้าหมวดหรือหน้ารายการสินค้า

สินค้า /customer/products

  • กรองตามหมวด / โปรโมชัน / เรียงลำดับ
  • พิมพ์ค้นหาชื่อสินค้า
  • เลือกหน้าแบ่งหน้าเมื่อมีสินค้าจำนวนมาก

รายละเอียดสินค้า /customer/products/:id

  • ดูรูป ราคา คำอธิบาย และระบุจำนวนเพื่อใส่ตะกร้า

ตะกร้า /customer/cart

  • ต้องล็อกอิน — แก้ไขจำนวน ลบรายการ ดูยอดรวม
  • ไปขั้นตอนเช็คเอาท์เมื่อพร้อมสั่ง

เช็คเอาท์ /customer/checkout

  • เลือกประเภทรับสินค้า (จัดส่ง / รับที่ร้าน ตามที่ระบบรองรับ)
  • เลือกหรือกรอกที่อยู่จัดส่ง
  • เลือกวิธีชำระเงินและยืนยันออเดอร์

รายการออเดอร์ /customer/orders

  • ดูทุกคำสั่งซื้อและสถานะปัจจุบัน
  • กดเข้าไปดูรายละเอียดแต่ละเลขที่

รายละเอียดออเดอร์ /customer/orders/:id

  • ดูรายการสินค้า ยอดเงิน สถานะ และช่องทางชำระเงิน
  • แนบหลักฐานการโอนเมื่อเลือกชำระแบบที่ต้องมีสลิป (ถ้าร้านเปิดใช้)

ติดตามการจัดส่ง /customer/tracking/:orderId

  • ดูความคืบหน้าคำสั่งซื้อที่อยู่ระหว่างจัดส่ง

โปรไฟล์ /customer/profile

  • แก้ข้อมูลส่วนตัวและที่อยู่จัดส่ง
  • ระบบอาจบังคับกรอกข้อมูลให้ครบก่อนใช้งานสั่งซื้อ — ทำในหน้านี้
  • คำขอเกี่ยวกับข้อมูลส่วนบุคคล (PDPA) ถ้ามีในหน้านี้

เข้าสู่ระบบ /customer/login

  • เข้าผ่าน LINE (ใน LINE WebView จะผูกบัญชีกับร้านอัตโนมัติตามโฟลว์ OAuth)

การเชื่อมกับ LINE

  • LIFF: เปิดเมนูร้านจากลิงก์ liff.line.me — ปุ่ม Rich Menu ควรชี้คนละหน้าเพื่อไม่ให้ทุกปุ่มไปหน้าแรกเหมือนกัน
  • แจ้งเตือน: เมื่อสถานะออเดอร์เปลี่ยน ลูกค้าที่ผูก LINE อาจได้รับข้อความและลิงก์เข้าดูรายละเอียดออเดอร์
  • รายละเอียดเทคนิคการตั้งค่า LINE Console และโฟลว์พารามิเตอร์ — ดูในไฟล์ LINE_Integration_Guide.md ในโปรเจกต์ และไฟล์ docs/line-flow-customer-offline.html สำหรับโฟลว์ให้ลูกค้าอ่านแบบสั้น

คนขับจัดส่ง

เข้าผ่านหน้าคนขับแยกจากลูกค้า — ใช้บัญชีคนที่ได้รับสิทธิ์จากแอดมิน

เข้าสู่ระบบ /driver/login

  • ยืนยันตัวตนตามที่ร้านกำหนด

แดชบอร์ด /driver/dashboard

  • ดูงานที่ได้รับมอบหมายและภาพรวมสถานะ

รายละเอียดงาน /driver/assignments/:assignmentId

  • ดูข้อมูลจัดส่ง ติดต่อลูกค้า และอัปเดตสถานะงานตามขั้นตอนที่ระบบให้

แอดมินร้าน (หลังบ้าน)

ใช้บนเบราว์เซอร์คอมพิวเตอร์ — ลงชื่อเข้าที่ /admin/login

ภาพรวม /admin/dashboard

  • สรุปตัวเลขและทางลัดไปฟังก์ชันหลัก

ออเดอร์ /admin/orders

  • ดูรายการคำสั่งซื้อ กรอง อัปเดตสถานะ ยืนยันสลิป (ถ้ามี) เปิดรายละเอียด
  • รายละเอียด /admin/orders/:orderId และติดตาม .../tracking

สินค้า /admin/products

  • จัดการรายการสินค้า เพิ่ม/แก้ไข /admin/products/new · /admin/products/:id/edit

หมวดหมู่ /admin/categories

  • จัดกลุ่มสินค้าสำหรับหน้าร้าน

โปรโมชันหน้าแรก /admin/home-promotions

  • ตั้งแบนเนอร์และลิงก์ที่ปรากฏบนหน้าแรกลูกค้า

ลูกค้า /admin/customers

  • ค้นดูและเปิดรายละเอียดลูกค้า /admin/customers/:customerId

บุคลากร /admin/personnel/staff · /admin/personnel/drivers

  • จัดการพนักงานและคนขับที่เข้าระบบหลังบ้านหรือแอปคนขับ

ตั้งค่าร้าน /admin/store-settings/...

  • ข้อมูลร้าน · การชำระเงิน · ค่าจัดส่ง · เวลาเปิด–ปิด

คลังสินค้า /admin/inventory/...

  • ภาพรวม · ปรับสต็อก · ผู้จัดจำหน่าย · ใบสั่งซื้อ · ความเคลื่อนไหวสต็อก
  • รายละเอียดใบสั่งซื้อ /admin/inventory/purchase-orders/:poId

บันทึกตรวจสอบ (Audit)

  • /admin/audit-log — ตรวจการกระทำสำคัญในระบบ (ถ้ามีสิทธิ์)
เอกสารสรุปจากโครงสร้างระบบในโปรเจกต์ — ฟังก์ชันบางอย่างอาจเปิดหรือปิดตามการตั้งค่าร้านและสิทธิ์ผู้ใช้จริง
